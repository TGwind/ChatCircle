# 创小圈校园资讯APP

该项目是一个校园资讯平台，为校园用户提供发帖、评论和接入ChatGPT多会话聊天等功能。此项目还在持续更新中，目前只实现部分功能。

后端服务：https://github.com/TGwind/Circle-SpringBoot.git

## 功能特点

### 用户管理
包括用户登录和注册功能。
### 发帖功能
用户可以发布校园资讯和帖子。
### ChatGPT接入
项目集成了ChatGPT，用户可以与ChatGPT进行对话交互
- 多会话系统
- 流式输出（打字机效果）
